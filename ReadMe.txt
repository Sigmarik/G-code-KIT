	Инструкция по исполькованию программы
		G-codeKIT
1) Общая информация:
	Программа представляет из себя визуализатор G-кода
	Язык - Python 3.7
	Необходимые для запуска файлы: GKIT.conf + arial.otf
	Использованные библиотеки: PyGame + math + sys
2) Как загрузить файл:
	Переместите файл с G-кодом в папку с программой.
	После этого можете запустить программу и ввести название вашего файла.
	Результат обработки файла будет лежать в файле LOG.txt
4) Управление
	Все подсказки по управлению отображаются на Вашем экране сразу после загрузки файла.
	Закрытие подсказок по управлению - [F1]
